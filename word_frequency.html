Words Count using Map - Shubham S. Yadav

Problem statement: Write a program to count words in a string/paragraph.
Language used: JavaScript
Features:
  • 1. Enter the text and the word to count the occurances of a specific word in that text.
  • 2. If word is not present in the text it will show not found.
  • 3. It is case insensitive.
